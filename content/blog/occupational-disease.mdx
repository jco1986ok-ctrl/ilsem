## 직업병이란?

"그날 사고가 아니라, 몇 년간 쌓인 질병."

일반 산재가 '돌발 사고' 중심이라면, 직업병은 **장기간 유해환경 노출**이 원인입니다.  
입증이 어렵지만, 최근 판례가 근로자에게 유리하게 바뀌고 있습니다.

---

## 대표 직업병 4가지 유형

### 1. 진폐증

| 항목 | 내용 |
|------|------|
| **발생 업종** | 채석·채탄·건설·터널·주물공장 |
| **원인 물질** | 석면, 규산, 암석 분진 |
| **인정 핵심** | 분진 노출 이력 증명 (근무 경력증명서, 작업환경측정 보고서) |
| **특별법** | 진폐예방법 별도 적용, 진폐심사위원회 심의 |

진폐증은 **진폐재해위로금** 별도 지급 (최대 약 1억 원)

### 2. 소음성 난청

| 항목 | 내용 |
|------|------|
| **발생 업종** | 제조공장, 건설, 광산, 사격장 |
| **인정 기준** | 85dB 이상 소음 노출 + 청력 손실 40dB 이상 |
| **입증 자료** | 작업환경측정 보고서, 청력검사 결과지 |

### 3. 직업성 암

| 항목 | 내용 |
|------|------|
| **대표 사례** | 석면 → 중피종, 벤젠 → 백혈병, 크롬 → 폐암 |
| **인정 난이도** | 높음 (역학적 인과관계 입증 필요) |
| **최근 판례** | "노출 기간·강도가 통상적 기준을 넘으면 업무관련성 인정" (대법원) |

### 4. 근골격계 직업병

| 항목 | 내용 |
|------|------|
| **대표 질환** | 수근관증후군, 경추간판탈출증, 요추염좌 |
| **인정 조건** | 반복적 동작, 부자연스러운 자세, 과도한 힘의 3요소 중 1개 이상 |
| **입증 자료** | 작업 동영상, 작업일지, 동료 진술서 |

---

## 직업병 입증 5대 자료

1. **작업환경측정 보고서** — 유해물질 농도·소음 수치
2. **건강검진 결과지** — 발병 전후 비교
3. **근무 경력증명서** — 노출 기간 입증
4. **동료 진술서** — 작업 환경 증언
5. **전문의 소견서** — 업무관련성 의학적 소견
